html속성 -요소에 대한 추가 정보를 제공 -모든 html요소는 속성을 가질 수 있다 -항상 시작 태그에 지정 -일반적으로 name="value"와 같은 이름="값" html 속성 사용시 주의사항 -항상 소문자 속성을 사용 -요소 이름 바로 다음에 나오는 속성은 요소 이름과 속성 사이에 공백이 필요하다 -하나 이상의 속성들이 있는 경우에는 속성 사이에 공백이 있어야 함 -속성 이름 다음에는 등호(=)가 붙는다 -속성 값은 열고 닫는 ' "," '로 감싸야 한다 HTML 속성의 종류 1.href 속성(hyper reference) 네이버 스포츠 2.src 속성(source: 소스코드) 3.alt 속성(altar: 바꾸다)
