블록 레벨 요소 vs 인라인 레벨 요소

블록 레벨 요소

:웹 페이지 상에서 블록을 만드는 요소

-앞뒤 요소 사이에 새로운 줄(line)을 만들고 나타냄 - - -

특징
인라인 요소에 중첩 불가능
다른 블록 레벨 요소에는 중첩 가능

인라인 레벨 요소

:문서의 한 단락같은 큰 범위에는 적용될 수 없다. 문장 단어 같은 작은 부분에서만 적용 가능 :필요한 만큼의 범위만 사용 :웹 페이지 택스트 내에서 사용되는 요소 :

인라인 레벨 요소 예시

인라인 요소는 새로운 줄(line)을 만들지 않는다


예시

단락 태크는 블록 레벨 요소 입니다

택스트나 구문을 꾸밀 때 사용하는 태그
